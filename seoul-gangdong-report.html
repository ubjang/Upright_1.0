PROJECT MOTIVATION

왜 '청렴 바이브' 웹앱을 기획했는가?

🚨 기존 방식의 핵심 Pain Point

  • 지루하고 하향식 교육 방식: 매년 형식적으로 시행되는 서면 위주, 주입식 청렴 교육으로 직원의 자발적인 동기부여가 크게 부족함.
  • 현실 실무와의 심각한 괴리: 구체적인 일상 실천보다 법령이나 교과서적이고 모호한 개념에 머물러 직원들이 마음에 와닿지 않음.
  • 참여 및 피드백 결여: 본인이 청렴도 향상을 위해 기여하고 있다는 체감이나 시각적인 성취 요소가 전혀 존재하지 않음.

✨ 청렴 바이브의 혁신적인 해결방안

  • 자율적 상호작용 플랫폼: 웹/모바일 친화적인 심플한 UX를 도입하여 언제 어디서나 단 1분 만에 가볍게 참여할 수 있도록 유도.
  • 일상 속 작은 습관의 적립: 매월 1일과 15일에 실천하기 쉬운 다짐 문구를 확인하고 퀴즈를 풀면서 자연스럽게 내재화.
  • '청렴 나무 성장' 메카니즘: 본인의 참여 이력이 누적되면서 새싹이 자라나 울창한 청렴 나무가 되는 과정을 시각화하여 성취감을 부여.
MVP CORE FUNCTIONALITY

현재 가동 중인 핵심 MVP 기능

1. 오늘의 청렴 실천 체크

공공기관 청렴도 평가의 6가지 핵심 축(직무윤리, 이해충돌, 금품·향응, 청탁·알선, 공정·투명, 행동강령)에서 추출된 다짐 문구를 보여줍니다. 서로 다른 평가 영역에서 추출된 3개의 실천 문구 중 본인이 오늘 다짐할 문구들을 편안하게 선택하여 약속합니다.

2. 사례 기반 마이크로 퀴즈

직무 현장에서 흔히 겪을 수 있는 미묘한 윤리적 갈등 상황을 OX나 객관식 형태의 시나리오 퀴즈(2문항)로 풀어봅니다. 답을 제출하는 즉시 알기 쉬운 핵심 해설이 나타나 복잡한 임직원 행동강령을 상황별로 학습할 수 있습니다.

3. 관리자 전용 관제 패널

지사 관리자는 암호화된 채널을 통해 직원 명부의 추가/수정/삭제를 진행할 수 있습니다. 또한, 지사 전체 직원의 날짜별 체크율, 퀴즈 정답률 현황을 요약된 수치 카드로 모니터링하여 추진 활동의 동향과 성과를 한눈에 파악합니다.

ARCHITECTURE & COMPLIANCE

시스템 아키텍처 및 철저한 익명성 보장

⚡ 최신 모던 웹 아키텍처

  • Next.js 16 (Turbopack) 기반의 초고속 페이지 로드 개발 생산성이 극대화되고 불필요한 번들 다운로드 없이 지사 직원들의 저사양 PC나 모바일 기기에서도 부드럽게 구동됩니다.
  • Tailwind CSS v4를 활용한 초경량화 스타일 코딩 성능을 저해하는 별도의 대형 CSS 라이브러리 없이 순수 유틸리티 스타일링을 통해 가벼운 인터페이스를 실현했습니다.
  • 서버 & 클라이언트 Supabase SDK 이원화 보안이 필요한 어드민 API와 일반 브라우저 화면의 DB 트랜잭션을 엄격하게 분류하여 설계 안정성을 높였습니다.

🔒 익명성 보장을 통한 자유로운 솔직 참여

  • Supabase Anonymous Sign-in (익명 세션 발급) 개인 사번이나 사내 ID를 기반으로 로그인할 때 발생하는 "내 정보를 사측에서 감시하는 것 아닌가?" 하는 심리적 저항선을 완벽하게 파괴합니다. 최초 접속 시 브라우저 단위로 고유한 임시 키가 발급되어 개인을 특정하지 못하도록 보장합니다.
  • Postgres Row Level Security (RLS) 강제 적용 데이터베이스 테이블 수준에서 다른 사용자의 제출 데이터를 무단으로 가로채거나 위조하지 못하도록 본인 세션에만 읽기/쓰기를 완벽 격리했습니다.
  • 보안 관리자 검증 절차 적용 직원 정보의 CRUD API 및 통계 관제 화면은 서버 환경 변수에 할당된 강력한 `ADMIN_PASSWORD` 인증을 거치지 않으면 비인가자가 절대로 해킹하거나 조회할 수 없도록 이중 방벽을 세웠습니다.
FUTURE ROADMAP

서울강동지사 청렴 바이브 고도화 로드맵

Phase 1 (현재 MVP 단계)

기본 청렴 지표 점검 및 나무 성장 시연

익명 가벼운 체크박스 서약 및 상황식 시나리오 퀴즈 2문항의 완벽 구현. Supabase RLS 테이블 기반의 이력 데이터 보안 저장 체계 완성 및 관리자 수동 직원 CRUD 가동.

Phase 2 (차기 릴리즈)

사내 매직링크(Magic Link) 인증 전환

보안 수준이 한 단계 완화된 기관 메일 계정으로 안전하게 접근할 수 있도록 1회용 매직링크를 도입하여 브라우저 쿠키 삭제 시 데이터 유실 문제를 극복하고 본인의 청렴 나무를 영구 보존합니다.

Phase 3 (지사 전파)

부서 단위 대항전 및 '강동 청렴의 숲' 론칭

지사 내 경영지원팀, 기술지원팀, 검사팀 등 부서별 실시간 참여율 데이터를 시각화하여 지사 전직원의 긍정적이고 즐거운 선의의 경쟁 구도를 구축합니다. 지사 내부 로비 모니터에 부서별 청렴 나무 숲을 미디어 아트로 실시간 전시하여 기관 대외 평가 준비에 기여합니다.
